# 京东项目说明:

1. 该项目使用了jq插件,      轮播图插件,   和 二级导航插件, 地理位置插件
2. 二级导航插件是自己仿写的, 通过这个 可以学会封装jq插件的方法,和思路
3. 在使用webpack打包的时候, 对于img图片资源, 可以使用require来引入
4. 实践了webpack打包工具的使用,对js ,less ,html ,img 的处理

### 项目使用到的技术:

1. webpack 自动化构建工具s
2. less
3. jquery
4. jquery插件
5. html+css 构建页面结构
6. 基本js动态交互

### 目录说明

1. plug-in 里面是封装的插件

   my-area 地理定位处的插件

   carousel 轮播图插件

   my-list 二级导航插件
